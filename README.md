# 🚀 TaskFlow - Professional MERN Stack Task Manager

A high-performance, production-quality Task Management application built using the **MERN Stack** (MongoDB, Express, React, Node.js). This project was designed to demonstrate clean code architecture, RESTful API design, and modern UI/UX principles.

## ✨ Key Features
- **Full CRUD Integration:** Create, read, update, and delete tasks with instant UI updates.
- **Dynamic Search:** Real-time search functionality to find tasks by title instantly.
- **Priority Filtering:** Advanced filtering system to categorize tasks by High, Medium, or Low priority.
- **State-of-the-Art UI:** Built with **Tailwind CSS** and **Lucide Icons** for a sleek, responsive dashboard experience.
- **Cloud Database:** Fully integrated with **MongoDB Atlas** for secure and persistent data storage.
- **Toast Notifications:** Real-time user feedback using **React-Toastify** for successful actions and error handling.

## 🛠️ Tech Stack

### Frontend
- **React.js (Vite)** - For a fast, component-based user interface.
- **Tailwind CSS** - For modern, utility-first professional styling.
- **Axios** - For seamless communication between Frontend and Backend.
- **Lucide React** - For high-quality, lightweight SVG icons.

### Backend
- **Node.js & Express.js** - Robust server-side environment and API routing.
- **Mongoose** - Elegant MongoDB object modeling for Node.js.
- **Dotenv** - For secure environment variable management.
